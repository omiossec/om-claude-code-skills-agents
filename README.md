# Claude Code Skills and Agent for Cloud OPS

In this repo, I will list skills and agent for Claude Code (May work with GitHub CLI) for Azure Cloud Operation. This include PowerShell, Azure PowerShell, Bicep, ARM Template, Azure Policy and Azure SRE Agent.

## SKILS

- PowerShell-Module, to build module with Claude Code
- bicep-best-practices, to create Azure Bicep template following best practices
- nxtools-dsc-linux, a skill to create DSC configurations for Linux
- azure-resource-graph, for Resource Graph KQL query
- azure-policy-assignment, for Azure Policy Assignmenent, remediation and examption
- azure-policy-defintion, for Azure Policy Definition

## Agent

- Security Code reviewer for PowerShell, pwsh-security-check.md
